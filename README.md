### 介绍

适用于第三方完全脱离盒子的 api-ms-win-crt-utility-l1-1-1.dll。
作用是获取 serverid 并向 TCP服务器 传递 id。
项目全程: "Ms-Win-Crt-Utility"
项目简称: "Mwcu"
它不包含任何验证代码，只负责传递信息。

### 支持说明
- java8 及以上版本
- Windows/Linux/MacOS
- 建议使用 Ninja 编译

### 说明
- launcherGameId = 登录后的服务器ID 【启动参数】
- userId = 登录后的用户ID 【启动参数】
